Detailed Description

A class for a MFT ladder. The ladder contains many AlpideChips (2,3,or 4). In addition to the array of chips and their number, the ladder class has an a variable that indicates whether the ladder is on the face or the back side of the disk so that the numbering of the columns can be set.
